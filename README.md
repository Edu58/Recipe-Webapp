# Recipe-Webapp
Recipe webapp made in HTML5, CSS3, vanilla javascript and the TheMealIDB API

![screenshot](screenshot.png)

The website doesn't use any framework and utilizes the fetch to request data from the API. It includes search functionlity and also loading of random recipes on window load.
